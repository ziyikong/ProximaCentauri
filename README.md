# Proxima Centauri

[![Ask DeepWiki](https://deepwiki.com/badge.svg)](https://deepwiki.com/128OTEML/ProximaCentauri)

A MOD originating from ice and snow for Mindustry.

## Features

- **New Planet**: Proxima - an ice-themed planet with custom terrain generation and 3D visuals
- **RBMK Nuclear Reactor System**: Inspired by real-world RBMK reactors with fuel rods, control rods, moderators, coolers, and a full control console
- **DFC Antimatter Core System**: Complete antimatter energy chain with core, stabilizers, injectors, energy emitters/receivers, and heat exchangers
- **Railway System**: Train tracks, stations, and factories for logistics
- **Custom Units**: Various ground, air, and naval units with unique abilities
- **Custom Effects**: Extensive visual effects including death effects, fragmentation, devastation, and more
- **Custom Rendering**: Custom shaders, 3D meshes, and rendering batches

## Building

### Desktop Testing
1. Install JDK **17**.
2. Run `gradlew jar`.
3. Your mod jar will be in the `build/libs` directory.

### Android Build
Use GitHub Actions CI to build for all platforms automatically.
